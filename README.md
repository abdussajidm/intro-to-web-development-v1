# Introduction to Web Development - Teaching Repository

This repository contains class material, practice tasks, examples, and mini-projects used to teach the fundamentals of web development.

The curriculum is beginner-friendly and covers:

- HTML
- CSS
- JavaScript
- Git, GitHub & Deployment basics
- Chrome Developer Tools
- Accessibility and other practical web development topics

## What This Repository Includes

- Daily class folders with agendas and examples
- Guided exercises and task files
- Solution folders for selected assignments
- Practice projects for real-world UI building
- Interview question notes and supporting resources

## Curriculum Focus

### 1) HTML Foundations

- Document structure
- Common elements and semantics
- Forms, tables, lists, and media
- Accessibility-friendly markup

### 2) CSS Foundations

- Selectors and specificity
- Box model
- Typography and color
- Layouts using Flexbox
- Pseudo-classes and practical styling patterns

### 3) JavaScript Foundations

- Variables, data types, and operators
- Conditions and loops
- Functions and basic DOM interactions
- Introductory scripting for interactive pages

### 4) Git, GitHub, and Workflow

- Repository setup and branching basics
- Commit and push workflow
- Collaboration practices
- Sharing project progress using GitHub

### 5) Deployment and Developer Tools

- Intro to deploying static websites
- Debugging and inspection with Chrome DevTools
- Performance and responsiveness checks
- Real-world workflow habits

## Repository Structure (High Level)

- `class/`: day-wise class agendas and in-class examples
- `projects/`: student tasks, solutions, and mini-project work
- `interview-questions/`: interview prep material
- `notes.md`, `discussions.txt`: class notes and discussion points

## How to Use This Repository

1. Start with the relevant day folder in `class/`.
2. Follow the `agenda.txt` for that day.
3. Open corresponding `.html` and `.css` files and practice by editing.
4. Use files in `projects/` for assignment-style practice.
5. Compare your result with solution folders when available.

## Recommended Setup

- VS Code & related extensions (Live Server, Prettier)
- Chrome browser
- Git installed locally
- GitHub account

Optional but helpful:

- VS Code Live Server extension
- Prettier extension for formatting

## Suggested Learning Flow

1. Learn concepts in class examples.
2. Practice with guided tasks.
3. Build mini components/pages.
4. Push your work to GitHub regularly.
5. Deploy small static projects and review in DevTools.

## Contribution Notes

This repo is primarily for teaching and practice. If you are contributing:

- Keep folder naming and structure consistent.
- Add clear file names for new examples.
- Keep examples simple, focused, and beginner-friendly.
