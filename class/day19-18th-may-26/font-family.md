# generic font families:

- serif
  - all fonts with serifs (small lines at the ends of characters) fall under this category.
    Examples include Times New Roman, Georgia, etc.

- sans-serif
  - all fonts without serifs fall under this category.
    Examples include Arial, Helvetica, etc.

- monospace
  - all fonts where each character occupies the same amount of horizontal space fall under this category.
    Examples include Courier New, Consolas, etc.

- cursive
  - all fonts that mimic handwriting fall under this category.
    Examples include Brush Script MT, Pacifico, etc.

- fantasy
  - all fonts that are decorative and do not fit into the other categories fall under this category.
    Examples include Impact, Jokerman, etc.
