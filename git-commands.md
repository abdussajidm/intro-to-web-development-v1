# git

- git is a version control system that allows developers to track changes in their code and collaborate with others. It was created by Linus Torvalds in 2005 for the development of the Linux kernel.

## git commands

- git init
- git add .
- git commit -m "first commit"
- git status
- git log
- git diff
- git branch
- git checkout -b branch_name
- git checkout branch_name
- git merge branch_name
- git remote add origin repo_url
- git push -u origin branch_name
- git pull origin branch_name
- git clone repo_url
- git stash
- git stash pop
- git reset --hard commit_id
- git revert commit_id
- git rm file_name
- git mv old_file_name new_file_name
- git tag tag_name
- git show tag_name
- git fetch
- git rebase branch_name
- git cherry-pick commit_id
- git log --oneline
- git log --graph --oneline --all
- git log --author="author_name"
- git log --since="date"
